## Features

- ✨ Onboarding com telas de boas-vindas e passos
- 🚀 Mapa com marcadores de mercados e localização do usuário
- ⚡ Filtro por categorias e listagem horizontal de locais
- 🎯 Tela de detalhes do mercado com cupons e informações
- 🔧 Consumo de API REST externa de categorias e mercados
